# 应付单查询API接口

## 1.接口说明

根据条件查询应付单

## 2.使用场景

根据条件查询应付单

## 3.接口调用说明

### 3.1请求类型

post

### 3.2请求示例

> http://127.0.0.1:8077/uapws/rest/jzyy/data/openapi

### 3.3请求URL参数说明

#### 3.3.1应付单查询条件信息

|参数|类型|是否必填|描述|
|------|------|------|------|
|funcType|String|请求类型|是|
|bill||||
|group_code|String|集团|是|
|org_code|String|销售组织|是|
|order_code|String|订单号|否|
|ct_code|String|合同号|否|
|invoice_code|String|发票号|否|
|cvendorid|String|供应商|否|
|usercode|采购员|String|
|dept_code|采购部门|String|
> 请求参数JSON示例

```
{
	"functype": "OA_PAYBILLQUERY",
	"bill": {
        	"cvendorid":"",
		"ct_code":"JZQ2020110B",
		"order_code": "CD2020101300000292",
		"invoice_code": "58447174-75",
		"org_code": "23",
		"group_code": "G",
                "dept_code": "",
		"usercode": ""
	}
}
```

### 3.4返回值说明
|参数说明|类型|描述|
|-------|---|----|
|code|状态码|String|201--成功 ，-1--失败|
|success|是否成功|String|true--成功 ，false--失败|
|message|提示信息|String|获取到8条数据|
|data_item||||
|vbillcode|应付单号|String|
|pk_payablebill|应付单表头ID|String|
|pk_payableitem|应付单表体ID|String|
|usercode|采购员|String|
|dept_code|采购部门|String|
|corigcurrencyid|币种|String|
|ninvoicemny|原币余额|String|
|cvendorid|供应商|String|
|vbatchcode|批次号|String|
|material|物料|String|
|project|项目|String|
|paymentdays|账期天数|String|
|pointsdays|启算日期|String|
