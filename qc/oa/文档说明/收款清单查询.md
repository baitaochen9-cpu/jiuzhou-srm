# 收款清单查询API接口

## 1.接口说明

根据条件查询收款单

## 2.使用场景

根据条件查询收款单

## 3.接口调用说明

### 3.1请求类型

post

### 3.2请求示例

> http://127.0.0.1:8077/uapws/rest/jzyy/data/openapi

### 3.3请求URL参数说明

#### 3.3.1收款单查询条件信息

|参数|类型|是否必填|描述|
|------|------|------|------|
|funcType|String|请求类型|是|
|bill||||
|group_code|String|集团|是|
|org_code|String|销售组织|是|
|vbillcode|String|单据号|否|
|dbilldate|String|收款日期|否|
|recaccount|String|收款银行账号|否|
|cashaccount|String|现金账户|否|
|payaccount|付款银行账号|String|
|currtype|币种|String|
|local_money_cr|金额|String|
|pk_balatype|结算方式|String|
> 请求参数JSON示例

```
{
	"functype": "OA_GATHERBILLQUERY",
	"bill": {
        "vbillcode":"D22022022100000179",
		"dbilldate":"2022-02-01,2022-06-11",
		"recaccount": "1102024809914017950",
		"cashaccount": "",
		"payaccount": "",
		"currtype": "CNY",
        "local_money_cr": "",
		"pk_balatype": "",
		"org_code": "23",
		"group_code": "G"
	}
}
```

|message|提示信息|String|获取到8条数据|
|data_item||||
|vbillcode|收款单号|String|
|pk_gatherbill|收款单表头ID|String|
|pk_gatheritem|收款单表体ID|String|
|usercode|采购员|String|
|dept_code|采购部门|String|
|corigcurrencyid|币种|String|
|ninvoicemny|原币余额|String|
|cvendorid|供应商|String|
|vbatchcode|批次号|String|
|material|物料|String|
|project|项目|String|
|paymentdays|账期天数|String|
|pointsdays|启算日期|String|
