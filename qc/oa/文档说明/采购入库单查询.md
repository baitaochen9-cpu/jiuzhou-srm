# 业务OPENAPI接口示例

## 1.接口说明
采购入库单据方案查询
## 2.使用场景
自定义查询方案查询采购入库单
## 3.接口调用说明

### 3.1请求类型 
post

### 3.2请求示例							
> http://127.0.0.1:8077/uapws/rest/jzyy/data/openapi

### 3.3请求URL参数说明
|参数|类型|是否必填|描述|
|------|------|------|------|
|funcType|String|请求类型|是|
|bill||||
|group_code|String|集团|是|
|org_code|String|销售组织|是|
|billdate|String|单据日期|是|
|dbizdate|String|入库日期|否|
|order_code|String|订单号|否|
|ct_code|String|合同号|否|
|vbillcode|String|入库单号|否|
|material|String|物料编码|否|

> 请求参数JSON示例
```
{
	"functype": "OA_ICBILLQUERY",
	"bill": {
        "cvendorid":"",
		"ct_code":"",
		"order_code": "",
		"vbillcode": "",
		"dbizdate": "2022-01-01 00:00:00~2022-06-01 00:00:00",
		"order_code": "",
		"org_code": "23",
		"group_code": "G",
		"billdate":"2022-01-01 00:00:00~2022-06-01 00:00:00",
		"material":"2310030001"
	}
}
```


### 3.4返回参数说明
#### 3.4.1 采购单的查询操作返回信息
|参数说明|类型|描述|
|-------|---|----|
|code|状态码|String|201--成功 ，-1--失败|
|success|是否成功|String|true--成功 ，false--失败|
|message|提示信息|String|获取到8条数据|
|data_item||||
|vbillcode|入库单号|String|
|cgeneralhid|入库单表头ID|String|
|cgeneralbid|入库单表体ID|String|
|cwarehouseid|仓库|String|
|usercode|采购员|String|
|dept_code|采购部门|String|
|corigcurrencyid|币种|String|
|ninvoicenum|可开票数量|String|
|ninvoicemny|可开票金额|String|
|cvendorid|供应商|String|
|vbatchcode|批次号|String|
|project|项目|String|
|manufacturer|生产厂商|String|
