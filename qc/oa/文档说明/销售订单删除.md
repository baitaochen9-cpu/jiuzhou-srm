# 销售订单删除

## 1.接口说明
查询
## 2.使用场景
销售订单删除
## 3.接口调用说明

### 3.1请求类型 
POST

### 3.2请求示例							
> http://127.0.0.1:8077/uapws/rest/jzyy/data/openapi

### 3.3请求URL参数说明
|参数|类型|描述|是否必填|
|------|------|------|------|
|funcType|String|请求类型|是|
|bill||||
|org_code|String|销售组织|是|
|vbillcode|String|单据号|是|
|ts|String|同步时间|是|
|Action|int|同步动作|是|


### 3.4 请求参数附加说明

> 请求参数JSON示例
```
{
	"functype": "OA_SALEORDER",
	"bill": {
		"org_code": "23",
		"vbillcode": "6301008611111",
		"Action": 1,
	
	}

}
```

### 3.4返回值说明
#### 3.4.1参数说明

**如返回为null，则显示为“空”**

|参数说明|描述|类型|说明|
|-------|----|----|-------|
|code|状态码|String|200--成功 ，-1--失败|
|success|是否成功|String|true--成功 ，false--失败|
|message|提示信息|String|删除销售订单成功|
|list||||
|code|单据号|String|Cd20211005007791|


#### 3.4.2返回值JSON示例 

```
{
	"message": "删除销售订单成功",
	"code": "200",
	"success": true,
	"list": {
		"code": "Cd20211005007791"
	}
}
```
